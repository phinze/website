# inze.ph

My personal website, built with Jekyll and self-hosted on [Miren](https://miren.dev).

Visit at: [inze.ph](https://inze.ph)

## Local Development

Run the site locally:

```bash
make serve
```

The site will be available at http://localhost:4000
